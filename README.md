# ETERNAL-TECHWORKS

A premium, modern multi-page web platform for **Eternal Techworks**—an industry-leading manufacturer of high-density PUF/PIR sandwich panels, rockwool insulation panels, and custom portable cabins.

## 🚀 Technologies Used
- **Frontend Core**: React 19, Vite, Javascript (ES6+)
- **Styling**: Tailwind CSS v4
- **Routing**: React Router v7
- **Icons**: Lucide React
- **Build Tool**: Vite

## ✨ Key Features
- **3D Isometric Panel Explode-Viewer**: An interactive, vector-based 3D model that lets users explode the sandwich panel layers (Outer Skin, Thermal Core, Inner Skin) to inspect technical specs, zinc thickness, and foam density.
- **Dynamic Spec Calculators**: Interactive tools on product pages for estimating thermal insulation and load-bearing capacity.
- **Lead Capture Form**: Responsive, UI-functional contact form for capturing customer inquiry leads.
- **Industrial Design System**: Clean, modern light theme layout tailored to industrial client trust.
